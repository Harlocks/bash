### Conditions Examples
### BASH

```sh
#!/bin/bash

if <comando <-> condicao>; then
  comando1
  comando2
fi
```

### OU

```sh
#!/bin/bash

if <comando <-> condicao>
then
  comando1
  comando2
fi
```

### OTHER EXAMPLES

```sh
#!/bin/bash

if <comando <-> condicao>
then
  comando1
  comando2
elif <comando <-> condicao>
then
  comando3
  comando4
else
  comando5
fi
```
